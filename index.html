csv-parser

description

this app service is used to parse and convert CSV file to JSON array.

endpoints

/api/upload
  • methods - post
  • payload - csv file
  • enctype - multipart/form-data
  • mimetype - text/csv or text/comma-separated-values
  • fieldname - csv

usage

send a post request to /api/upload, the fieldname of the uploaded file must be csv and the column headers must be name, organization, award, date, description, certificate_number

example usage

visit /api/upload/test, upload a csv file with the following heading:

  • name
  • organization
  • description
  • award
  • date
  • certificate_number

view returned JSON array
